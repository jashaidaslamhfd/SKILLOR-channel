# Conversion France-first

Les composants de génération, catalogue, SEO, sous-titres, voix, upload et workflow GitHub Actions ont été configurés pour le marché France / francophonie. La voix de production est désormais Kokoro français (`ff_siwis`), et les sous-titres envoyés à YouTube sont étiquetés `fr`.

L'upload reste privé par défaut. Testez une vidéo dans YouTube Studio avant toute publication publique.
